## 🧩 Recent Work

**[KinPT](https://github.com/kin-pt/kin-pt)**: AI-powered Android telerehabilitation app. Uses on-device computer vision (MediaPipe BlazePose) to track 33 body landmarks in real time, providing live skeleton overlay, rep counting, and audio/visual form feedback; no equipment, no cloud upload, and an elderly-friendly UI. Senior Design team of 6 where I served as Project Manager and Frontend Lead.  
`Flutter` · `Kotlin` · `MediaPipe` · `Firebase` · `Node.js`

**[Brain Network Analyzer](https://github.com/soleil-cordray/brain-network-analysis)**: Solo graph-theoretic analysis of a 90-region fMRI connectivity network. Applied Edmonds-Karp max-flow, four centrality measures, spreading activation, and modularity-based community detection. Modeled depression, anxiety, and ADHD as edge-weight perturbations grounded in published neuroscience literature.  
`Python` · `NetworkX` · `NumPy` · `Nilearn`

**[Malware Behavior Comparison](https://github.com/soleil-cordray/malware-behavior-comparison)**: Ghidra static analysis of a real Remcos RAT binary alongside fileless and living-off-the-land samples. Uncovered dynamic API loading, registry-based persistence, and an embedded C2 URL; mapped findings to MITRE ATT&CK and built a behavioral classification framework across the three categories.  
`Ghidra` · `VMware` · `MITRE ATT&CK`

**[Parallelized Sudoku Solvers](https://github.com/soleil-cordray/parallelized-sudoku-solvers)**: Benchmarked three algorithms (Brute Force, Dancing Links, Propagation-Cross-Search) in sequential and parallelized forms across 9×9, 16×16, and 25×25 boards with easy, medium, and hard difficulties. Found that thread overhead often outweighed speedup; Propagation-Cross-Search was fastest overall.  
`Java` · `C++` · `pthreads` · `ExecutorService`

**[GameOn](https://github.com/soleil-cordray/game-on)**: Collaborative web platform with a real-time WPM typing game and color reaction game, plus user accounts and a leaderboard. Built the WPM game from scratch (keystroke tracking, accuracy calculation, live timer) and contributed to authentication. Currently being polished and re-hosted.  
`React` · `JavaScript` · `MongoDB` · `Bootstrap`
